Información sobre Tipos de File Systems FSx
Amazon FSx for Lustre
  • Optimizado para computación de alto rendimiento (HPC)
  • Compatible con cargas de trabajo de machine learning
  • Integración con S3 para datos
  • Escalado automático de throughput
Amazon FSx for Windows File Server
  • Sistema de archivos nativo de Windows
  • Compatibilidad total con SMB
  • Integración con Active Directory
  • Capacidades de DFS y deduplicación
Amazon FSx for NetApp ONTAP
  • Capacidades avanzadas de gestión de datos
  • Snapshots, clones y replicación
  • Compatibilidad con protocolos NFS, SMB e iSCSI
  • Compresión y deduplicación
Amazon FSx for OpenZFS
  • Sistema de archivos con snapshots instantáneos
  • Clones de bajo costo y alta eficiencia
  • Compresión de datos integrada
  • Compatible con protocolos NFS
