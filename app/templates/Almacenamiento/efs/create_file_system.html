Información sobre File Systems EFS
¿Qué es Amazon EFS?

Amazon Elastic File System (EFS) es un sistema de archivos NFS simple, escalable y elástico para usar con servicios de AWS Cloud y recursos locales. Está diseñado para ser altamente disponible y duradero.

Modos de Rendimiento
  • General Purpose: Para la mayoría de las cargas de trabajo
  • Max I/O: Para aplicaciones que requieren el mayor throughput posible
Modos de Throughput
  • Bursting: Escala automáticamente según el uso
  • Provisioned: Throughput garantizado (con costo adicional)
Mount Targets

Después de crear el file system, necesitarás crear mount targets en las subnets donde quieras acceder al sistema de archivos desde instancias EC2.
