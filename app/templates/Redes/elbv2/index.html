Elastic Load Balancing (ELB)

ELB distribuye automáticamente el tráfico de aplicaciones entrante entre múltiples destinos, como instancias EC2, contenedores y direcciones IP.

Tipos de Load Balancers:

  • Application Load Balancer (ALB): Enruta basado en contenido HTTP
  • Network Load Balancer (NLB): Enruta basado en direcciones IP y puertos
  • Gateway Load Balancer (GLB): Para appliances virtuales de terceros
  • Classic Load Balancer: Generación anterior (legacy)
