Información del Servicio

AWS Systems Manager proporciona una interfaz unificada para configurar y administrar recursos de AWS y en las instalaciones. Incluye herramientas para:

  • Parameter Store: Almacenamiento seguro de parámetros de configuración
  • Run Command: Ejecución remota de comandos en flotas de instancias
  • Session Manager: Administración de sesiones interactivas seguras
  • State Manager: Aplicación automática de configuraciones deseadas
  • Inventory: Recopilación de metadatos sobre instancias y software
