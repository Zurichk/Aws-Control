CloudTrail Insights

Métricas avanzadas y análisis de comportamiento inusual

Acerca de CloudTrail Insights

CloudTrail Insights utiliza machine learning para detectar actividad inusual en la cuenta AWS. Identifica automáticamente:

  • Actividad API inusual: Cambios significativos en el volumen de llamadas a API
  • Errores de API inusuales: Aumento anormal en errores de llamadas a API
  • Eventos de gestión inusuales: Actividad inusual en operaciones de gestión
  • Patrones de comportamiento: Detección de anomalías en el comportamiento normal de la cuenta
Estado de Insights

CloudTrail Insights está disponible para trails que cumplan con los requisitos mínimos.

Requiere al menos 30 días de historial de eventos para generar insights.
Beneficios de Seguridad

Mejora la detección de amenazas y actividades sospechosas en tiempo real.

  • Detección automática de anomalías
  • Alertas proactivas
  • Integración con servicios de seguridad
Métricas de Insights Disponibles
Volumen de API

Cambios significativos en el número de llamadas a API por hora

Tasa de Errores

Aumento anormal en errores de llamadas a API

Actividad de Gestión

Cambios inusuales en operaciones de gestión de recursos

Configuración de Insights
Requisitos para Insights
  • Trail activo por al menos 30 días
  • Historial suficiente de eventos
  • Trail configurado correctamente
  • Permisos adecuados en IAM
Mejores Prácticas
  • Habilitar insights en trails críticos
  • Configurar alertas para insights importantes
  • Revisar insights regularmente
  • Integrar con servicios de monitoreo
Costos de CloudTrail Insights

CloudTrail Insights tiene costos adicionales:

  • $0.35 por 100,000 eventos analizados para insights
  • Costo único por insight generado (alrededor de $0.10-0.50 por insight)
  • No hay costo si no se generan insights
  • Período de evaluación gratuito durante los primeros 30 días
