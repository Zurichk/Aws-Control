Acerca de Amazon SageMaker

Amazon SageMaker es un servicio completamente administrado que proporciona a cada desarrollador y científico de datos la capacidad de construir, entrenar e implementar modelos de machine learning (ML) rápidamente. SageMaker elimina el trabajo pesado de cada paso del proceso de machine learning para facilitar el desarrollo de modelos de alta calidad.
