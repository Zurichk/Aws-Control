¿Qué es un Endpoint?

Un endpoint de SageMaker es un servicio web HTTPS que permite hacer predicciones en tiempo real usando un modelo de machine learning desplegado.

Requisitos:
  • Configuración de endpoint existente
  • Modelo entrenado disponible
  • Permisos IAM adecuados
Costo:

Se cobra por hora de instancia mientras el endpoint esté activo.
