Información sobre Versiones
¿Qué es una versión?

Una versión es una instantánea inmutable del código y configuración de una función Lambda. Una vez creada, no puede modificarse.

$LATEST vs Versiones
  • $LATEST: Versión editable del código
  • Versiones numeradas: Instantáneas inmutables
  • Aliases: Punteros a versiones específicas
