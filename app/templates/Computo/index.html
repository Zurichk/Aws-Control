Amazon Elastic Compute Cloud (EC2)

EC2 es el servicio de computo en la nube de AWS que proporciona capacidad de computo escalable. Te permite lanzar instancias de servidor virtual en minutos, con una amplia variedad de tipos de instancia optimizados para diferentes casos de uso.

¿Qué puedes hacer con EC2?

  • Lanzar instancias: Crea servidores virtuales con diferentes configuraciones de CPU, memoria y almacenamiento.
  • Escalado automático: Ajusta automáticamente el número de instancias según la demanda.
  • Balanceo de carga: Distribuye el tráfico entre múltiples instancias para alta disponibilidad.
  • Almacenamiento persistente: Usa EBS para almacenamiento de datos que persiste más allá del ciclo de vida de la instancia.
