Amazon DocumentDB

Amazon DocumentDB es un servicio de base de datos de documentos completamente administrado que es compatible con MongoDB. Proporciona la escalabilidad, durabilidad y seguridad que necesitas para operar aplicaciones sensibles a la escala de producción.

Características principales de DocumentDB:

  • Compatible con MongoDB: API compatible con MongoDB 3.6, 4.0 y 5.0
  • Alto rendimiento: Arquitectura distribuida con réplicas de lectura
  • Escalabilidad: Hasta 64 TB de almacenamiento, hasta 15 réplicas de lectura
  • Durabilidad: Almacenamiento en 6 copias en 3 zonas de disponibilidad
  • Seguridad: Encriptación en reposo, VPC, IAM, KMS
