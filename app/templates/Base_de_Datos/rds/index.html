Acerca de Amazon RDS

Amazon RDS facilita la configuración, el funcionamiento y el escalado de una base de datos relacional en la nube. Proporciona capacidad rentable y redimensionable al mismo tiempo que automatiza las tareas de administración de bases de datos que requieren mucho tiempo, lo que te permite concentrarte en tus aplicaciones.
