Amazon Neptune

Amazon Neptune es un servicio de base de datos de gráficos rápido, confiable y completamente administrado que facilita la creación y ejecución de aplicaciones que funcionan con conjuntos de datos altamente conectados.

Características principales de Neptune:

  • Base de datos de gráficos: Optimizada para almacenar y navegar relaciones complejas
  • Motores soportados: Apache TinkerPop Gremlin y SPARQL
  • Alto rendimiento: Diseñado para consultas de gráficos complejas con latencia baja
  • Escalabilidad: Hasta 64 TB de almacenamiento, réplicas de lectura
  • Seguridad: Encriptación en reposo y en tránsito, integración con IAM
