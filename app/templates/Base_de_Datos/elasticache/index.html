Amazon ElastiCache

Servicio de Cache en Memoria de Alto Rendimiento

Amazon ElastiCache es un servicio web que facilita la implementación, el funcionamiento y el escalado de almacenes de datos en memoria en la nube. Mejora el rendimiento de las aplicaciones web al permitirle recuperar información de almacenes de datos en memoria rápidos y administrados, en lugar de depender únicamente de almacenes de datos basados en disco más lentos.

Motores Soportados:
  • Redis: Almacén de datos en memoria de código abierto
  • Memcached: Sistema de cache de objetos de memoria distribuida
Casos de Uso
  • • Cache de aplicaciones web
  • • Almacenamiento de sesiones
  • • Leaderboards y rankings
  • • Cache de base de datos
  • • Análisis en tiempo real
