Acerca de Amazon Kinesis
¿Qué es Kinesis?

Amazon Kinesis es una plataforma para streaming de datos en la nube de AWS. Permite recopilar, procesar y analizar datos de streaming en tiempo real.

Características principales:
  • Procesamiento en tiempo real
  • Escalabilidad automática
  • Integración con otros servicios AWS
  • Almacenamiento duradero
