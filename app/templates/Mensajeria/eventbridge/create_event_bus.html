¿Qué es un Event Bus?

Un Event Bus es un canal que recibe eventos de diferentes fuentes y los distribuye a diferentes destinos basándose en reglas.

Event Bus por defecto:
  • Ya existe en todas las cuentas AWS
  • Recibe eventos de servicios AWS
  • No puede ser eliminado
Event Buses personalizados:
  • Permiten segmentar eventos
  • Aislan dominios de aplicación
  • Pueden tener políticas de acceso
